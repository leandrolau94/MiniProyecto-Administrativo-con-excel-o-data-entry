# MiniProyecto-Administrativo-con-excel-o-data-entry
Un miniproyecto de administrativo con excel o dataentry para mostrar habilidades en el puesto en cuanto a añadir nuevos datos o borrar, ordenar y filtrar, limpieza de datos, Revision de Errores y funciones clave para trabajo autoimatizado en excel o libreoffice
## Herramientas usadas <br> * Añadir nuevos datos <br> * Ordenar y filtrar <br> * Limpieza de datos y revision de errores con ayuda de las herramientas formato condicional o filtro standard (control de calidad a traves de la deteccion y eliminacion de duplicados, etc) <br> * Uso de las funciones SI(IF), SUMAR(SUM), SUMAPRODUCTO(SUMPRODUCT), BUSCARV(VLOOKUP) <br> * Uso de tablas dinamicas para analisis de venta

# 📊 Sales Analytics Dashboard (Power BI)

## 📌 Overview

In addition, this project includes a **Business Intelligence dashboard** built with Power BI to analyze sales performance.

---

## 🎯 Objectives

* Analyze sales data by product, customer and time
* Provide key business insights through interactive visualizations
* Support decision-making using KPIs

---

## 🔧 Features

* Interactive dashboard with slicers (date, product, customer)
* Key KPIs:

  * Total Revenue
  * Total Quantity
  * Average Ticket
  * Number of Customers
  * Number of Products
* Dynamic filtering and real-time updates
* Clean and professional layout

---

## 📊 Key Insights

* Identification of top-performing products
* Customer segmentation based on purchase behavior
* Sales trends over time

---

## 🛠️ Tools Used

* Power BI
* DAX (Data Analysis Expressions)

---

## 📁 Dataset

The dashboard is based on a structured sales dataset including:

* Date
* Product
* Customer
* Quantity
* Price
* Total Revenue

---

---

## 🚀 How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Use slicers to filter data dynamically
3. Explore KPIs and visual insights
